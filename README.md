## About This Project
This repository contains a web-based presentation for the Ujima SACCO Ethical Architecture Dossier. The project outlines the transition from an "ethically blind" AI lending model to a "dignity-centered" architecture designed to respect human agency, local economic realities, and regulatory frameworks.

This dossier was developed as an academic assignment focusing on AI ethics, algorithmic fairness, and data sovereignty in financial technology.

## Core Frameworks Explored

* **Dignity-Centered Design:** Embedding moral guardrails (ETHOS) to evaluate informal cash flows without penalizing vulnerable demographics (e.g., informal traders).
* **TRACK Audit:** * **T**raining Data & **R**epresentation: Correcting historical urban-centric biases.
  * **A**mplification, **C**ounterfactuals, & **K**ill Switches: Ensuring algorithmic fairness.
* **OASIS Data Sovereignty:** Ensuring compliance with the Kenya Data Protection Act 2022 (ODPC) through local storage (AWS Africa), strict opt-in consent, and k-anonymity.
* **PRIDE Loop Governance:** Human-in-the-loop calibration, interpretability standards, and the establishment of an Elders Council for oversight.
